# 🏠 Kostrakan App (Smart Boarding House System)

Aplikasi pencarian dan manajemen kost berbasis **mobile (Kotlin)** dan **web (Vue.js)** yang terintegrasi dengan **backend API (Node.js)**.
Sistem ini membantu **penyewa menemukan kost terdekat** menggunakan perhitungan jarak **Haversine**, serta membantu **pemilik kost mengelola properti mereka**.

---

## 🚀 Features

### 👤 Penyewa (Mobile - Kotlin)

* 🔍 Cari kost berdasarkan lokasi
* 📍 Rekomendasi kost terdekat (Haversine)
* 🏠 Detail informasi kost
* 📅 Booking / reservasi kost

### 🏢 Pemilik Kost (Web - Vue.js)

* 🏠 CRUD data kost
* 📸 Upload gambar kost (Cloudinary)
* 📊 Manajemen data penyewa
* 📅 Monitoring booking

### ⚙️ Backend API

* 🔐 Authentication (JWT)
* 🗄️ Manajemen data kost & user
* ☁️ Upload image (Cloudinary)
* 📍 Perhitungan jarak (Haversine)
* 🔄 RESTful API

---

## 🧱 Tech Stack

### 📱 Mobile (Penyewa)

* Kotlin (Android Native)

### 💻 Web (Pemilik)

* Vue.js

### ⚙️ Backend

* Node.js (Express.js)

* JWT Authentication
* Cloudinary (Image Upload)
